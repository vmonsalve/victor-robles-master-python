# Master de Python por Victor Robles

## Temario:

1. Programación desde cero y paso a paso, sin necesidad de tener conocimientos previos.

2. Python, uno de los lenguajes de programación más populares en la actualidad.

3. POO, Programación Orientada a Objetos en Python.

4. Bases de datos SQL, trabajando en conjunto con nuestros desarrollos.

5. Módulos y frameworks, para dar el salto profesional con Python.

6. Tkinter, para crear aplicaciones de escritorio con interfaz gráfica.

7. Desarrollo web, creando varios proyectos web.

8. Django, el framework de desarrollo web para Python más popular y demandado por las empresas.

9. Flask, el moderno framework para desarrollar aplicaciones web.

10. Habilidades full-stack, haciendo cientos de ejercicios, prácticas y proyectos completos.
